Descrição

Num futuro distante, forças rivais batalham entre si usando robôs gigantes, guerreando pelo controle dos valiosos recursos de solarite. Na posição de comandante de um sentinela, você deve travar guerras com aqueles que cruzarem o seu caminho, completar missões, ganhar solarite, adquirir novas tecnologias e usá-las para fazer com que seu sentinela seja o mais temível da galáxia.

  4. Como posso melhorar minha pontuação?
  5. O que é perda de energia?

1. Posso mudar os controles de câmera?

Sim. No menu de opções você pode definir várias configurações. Experimente cada uma delas para descobrir qual é a melhor para você.

2. O que devo fazer para desbloquear mais tecnologias?

Jogue campanhas para um jogador para desbloquear muitas tecnologias iniciais. Você também pode desenvolver tecnologia direto da tela de "Configuração" ao gastar solarite adquirida em jogos qualificatórios.

3. Como consigo novos sentinelas?

Para poder usar um novo chassi, você deve desenvolver uma série de componentes e módulos padrão de cada sentinela. Novos chassis ficam disponíveis a medida que você progride tecnologicamente.

4. Como posso melhorar minha pontuação?

Cause mais danos e extermine mais inimigos e, claro, tente não causar danos aos seus companheiros de equipe.

5. O que é perda de energia?

Usar a propulsão, a mira e armas energéticas gastam a energia do seu sentinela. Sua energia vai recarregar com o tempo, mas se usá-la rapidamente, seu sentinela vai descarregar completamente e ficará imóvel até que ele se recupere. Evite a perda de energia usando mais capacitores e sendo econômico com sua energia.

6. Como faço para colecionar medalhas?

As medalhas são concedidas aos comandantes que fizerem um esforço extraordinário no campo de batalha. Qualquer medalha usada em jogos qualificatórios são registradas e o total das medalhas acumuladas por você podem ser vistas na página de Conquistas.

7. Quais são as especificações mínimas para este jogo?

As especificações mínimas recomendadas para este jogo são:

  • 500 MHz
  • 128 MB de RAM

As especificações recomendadas para aproveitar ao máximo este jogo são:

  • 1.5 GHz
  • 256 MB de RAM

Abaixo você encontrará uma explicação dos termos das nossas especificações técnicas.

Megahertz (MHz) / Gigahertz (GHz) é uma medida que define a velocidade do processador do computador. Isso afeta a rapidez com que você pode executar jogos e aplicativos, por exemplo.

Os Megabytes (MB) são usados para medir a quantidade de memória do computador. Isso afeta a rapidez com que o computador é executado ao usar programas grandes, como jogos, ou quando você executa vários programas ao mesmo tempo.
