Login múltiplo

Data de vigência: 12 de maio de 2009

Você pode criar mais de uma conta, mas, se fizer isso, não poderá fazer logon em mais de uma conta em nenhum momento e elas não devem interagir com outras de nenhuma maneira. Isso inclui ‘negociação de itens abandonados’ ou qualquer outro método de transferência de itens.

Questões relacionadas
Abaixo estão algumas questões relacionadas que podem ajudar a entender e seguir esta regra.

  1. Por que temos esta regra?
  2. Posso usar um software de terceiros para fazer login em uma conta para mim enquanto estou conectado a outra (para que eu pessoalmente utilize somente uma conta)?
  3. Tenho duas contas. Um amigo meu pode jogar com uma para que eu então possa fazer trocas entre os personagens?
  4. Posso dar itens de minha conta para outro jogador, para que ele possa dá-los a outro personagem com o qual eu jogo?
  5. Tudo bem se eu trocar objetos no jogo com alguém que mora comigo?

1. Por que temos esta regra?

Temos esta regra para manter uma experiência de jogo justo. Não queremos que nossos jogos se tornem jogos onde todos sentem que necessitam usar múltiplas contas de uma só vez para competir. É bem melhor juntar-se a seus amigos.

Voltar ao topo
2. Posso usar um software de terceiros para fazer o login em uma conta para mim enquanto estou conectado a outra (para que eu pessoalmente esteja usando somente uma conta)?

Não, você não deve usar software de terceiros para obter uma vantagem desleal - consulte uso de macros, bots or softwares de terceiros.

Voltar ao topo
3. Tenho duas contas. Um amigo meu pode jogar com uma para que eu então possa fazer trocas entre os personagens?

Não, isso é contra esta regra. Você não está autorizado a trocar de personagens pertencentes a um jogador. Além disso, você estaria violando a regra de Compartilhamento/Troca de Contas.

Voltar ao topo
4. Posso dar itens de minha conta para outro jogador, para que ele possa dá-los a outro personagem com o qual eu jogo?

Não, isso é contra esta regra. Você não deve transferir itens entre suas contas de nenhuma maneira.

Voltar ao topo
5. Tudo bem se eu trocar objetos no jogo com alguém que mora comigo?

Sim, isso não viola as Regras de Conduta desde que vocês estejam jogando com duas contas separadas criadas por cada um de vocês.

Para que o RuneScape continue sendo apreciado por todos, favor observar e seguir nossas regras. Não toleraremos jogadores que incomodam e, se você violar nossas regras, provavelmente sua conta receberá uma suspensão ou restrição. Membros suspensos ou restringidos por violarem as nossas regras não têm direito a nenhum tipo de reembolso.
