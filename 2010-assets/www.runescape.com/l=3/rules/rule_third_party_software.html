Usar macros e usar bots ou software de terceiros

Data de vigência: 12 de maio de 2009

Você não deverá utilizar outro software para obter uma vantagem desleal em nossos jogos. Isso inclui ferramentas de automação, macros, programas bots ou digitadores automáticos. Você não pode evitar o uso de quaisquer de nossos mecanismos projetados para efetuar logout automaticamente de usuários inativos.

Além disso, você não pode utilizar nenhum software de terceiros de jogos específicos que incentive a violação de nossas regras ou se desvie da navegação normal de nosso site, ou que solicite automaticamente páginas de nossos sites ou que ponha em risco contas de usuários.


Perguntas relacionadas

Veja abaixo algumas perguntas que podem ajudar a compreender melhor e seguir esta regra:

  1. Por que temos esta regra?
  2. Que tipo de software de terceiros NÃO é permitido?
  3. O que vocês querem dizer com "software"?
  4. Como funciona o sistema de detecção de uma nova macro?
  5. Será possível apelar se eu for suspenso por utilizar macros?
  6. O que é o Treinamento AFK?

1. Por que temos esta regra?

Primeiramente, não queremos jogadores fazendo download de software de terceiros a partir de fontes não confiáveis que podem conter, e frequentemente contêm, keyloggers, cavalos-de-tróia e vírus. Muitos jogadores tiveram suas contas roubadas porque instalaram inadvertidamente um software de hacker em suas máquinas. Mesmo que o autor tenha tido a melhor das intenções, uma falha inesperada no software poderia ainda colocar sua conta em risco. Até mesmo alguns softwares supostamente de boa reputação tiveram problemas no passado. Há também casos de pessoas que criaram deliberadamente softwares que aparentemente eram um ótimo complemento (add-on), mas que na verdade apenas roubavam sua senha.

Em segundo lugar, as macros arruinam o jogo para todos. Não queremos que alguns jogadores tenham uma vantagem desleal e preencham todas as melhores marcas do treinamento com programas bots. Este jogo foi projetado para ser apreciado por humanos, não por programas bots. No passado, quando não exigíamos o cumprimento desta regra, descobrimos rapidamente que isso estragava o jogo e os próprios jogadores começaram a exigir a imposição desta regra. Agora, exigimos o total cumprimento desta regra.

2. Que tipo de software de terceiros NÃO é permitido?

Isso inclui, mas não se limita aos seguintes softwares de terceiros:

  1. Software que tente se comunicar diretamente com nossos universos de jogos.
  2. Software que monitore, modifique ou decodifique as comunicações entre nossos miniaplicativos para jogos e nossos universos de jogos.
  3. Software que gere inserção de dados em nossos miniaplicativos para jogos. Isso inclui software que mova automaticamente o ponteiro do mouse ou gere cliques do mouse ou pressione teclas.
  4. Versões modificadas ou substituídas de nossos miniaplicativos para jogos. É estritamente proibida a engenharia reversa de nossos miniaplicativos para jogos. Não é permitida por nenhuma razão a utilização de uma versão modificada de nossos miniaplicativos para jogos. Software de terceiros que exibam páginas/conteúdo de nosso(s) site(s) e que tenham recursos específicos (isto é, caixa de ferramentas) não deve ser utilizado para acessar nosso site, a menos que esse software cumpra todas as outras regras, assim como as seguintes regras adicionais:
    • Não deve incentivar violações de regras ou pôr em risco os usuários.
    • Não deve conter anúncios ou links para sites que incentivam a violação de regras ou ponham em risco os usuários.
    • Não deve fazer automaticamente solicitações repetidas de página/conteúdo de nossos sites ou de outra forma fazer solicitações intensivas de largura de banda de modo excessivo (cada solicitação de página/conteúdo deve ser somente em resposta direta a uma solicitação de usuário de cada vez).
    • Não deverá evitar a navegação normal de nosso site para usuários que desejam chegar aos universos dos jogos. O software deverá inicialmente apenas indicar a primeira página de nosso site e solicitar ao usuário que navegue pelo site normalmente para chegar aos universos dos jogos. O software não deverá conter ou adquirir links diretos para os aplicativos do jogo.
    • Não deve ocultar ou incentivar a ocultação de quaisquer anúncios em nosso site. Para evitar incentivar os usuários a ocultar o anúncio acima do jogo, as páginas de exibição da tela de nosso site devem estar em um padrão de pelo menos 630 pixels (se visualizadas em uma tela com resolução superior a 800x600), e não deve ser possível redimensioná-la a uma altura inferior a essa, isto é, a página deverá ser exibida de tal maneira que tanto o jogo quanto o anúncio estejam completamente visíveis ao mesmo tempo.
    • Não deve tentar tornar a entrada em um mundo concorrido de jogo mais fácil do que seria para usuários que não estejam utilizando o software. Não deve conter quaisquer recursos para facilitar a entrada ou monitorar mundos concorridos.
    • Se o software tiver seu próprio recurso de bate-papo integrado, separado de um bate-papo de jogo, deve esclarecer aos usuários que esse bate-papo não faz parte de nossos jogos e não é oficial, endossado ou moderado pela Jagex de maneira alguma. Além disso, esse bate-papo deve ainda ser moderado (mas não por nós) e os moderadores não devem permitir bate-papos que violem nossas regras de conduta. A fim de evitar confusão com o bate-papo oficial do jogo, os moderadores do bate-papo externo não devem ser identificados com uma coroa de prata ou de ouro. Se o bate-papo for baseado em IRC, o software deve somente permitir o acesso aos canais IRC que seguirem as regras acima.

Em qualquer caso, o uso desse software não é recomendado. Não recomendamos o uso de NENHUM software de terceiros, pois talvez sejam extremamente perigosos. Por favor, esteja ciente de que existem softwares externos que declaram ser um ótimo complemento (add-on), mas de fato instalam um keylogger no computador e roubam sua senha. Você também deve ter ciência de que existem softwares perigosos que não necessariamente serão identificados por seu software antivírus. Fazer uma varredura nos dowloads com certeza é uma ótima idéia, mas ainda assim não elimina o risco totalmente.

3. O que vocês querem dizer com "software"?

Software significa qualquer tipo de programa, inclusive: softwares que podem ser descarregados (por exemplo, arquivos .exe) ou softwares com base na Internet (por exemplo, um site configurado para realizar uma função similar com flash, java, javascript ou serverside scripts).

4. Como funciona o sistema de detecção de macros?

Nosso sistema de detecção de macros monitora a atividade em nossos servidores e detecta comportamentos suspeitos. As evidências são analisadas por nossa equipe de especialistas que confirmam ou recusam a análise do sistema. A Jagex não revelará os mecanismos exatos sobre como esse sistema funciona, porque isso facilitaria fraudes.

5. Será possível apelar se eu for suspenso por utilizar macros?

Nossa política sempre foi a de não tolerar nenhuma forma de fraude. Não suspendemos contas de modo leviano. Evidências significativas são sempre coletadas antes que qualquer medida seja tomada contra um jogador. Não revelaremos as evidências coletadas, já que a maioria das pessoas que fraudam somente desejam conhecer as evidências para evitar que sejam pegos da próxima vez. Você pode enviar uma apelação, visitando a página Gerenciamento de conta e selecionando "Apelação de ofensa".

6. O que é o Treinamento AFK?

O Treinamento AFK (do inglês Away From Keyboard) é um treinamento "Distante do Teclado". Enquanto estiver jogando nossos jogos, você deverá estar em um computador. O treinamento "distante do teclado" não é permitido e você deverá efetuar logout quando sair do computador.

Para que o RuneScape continue sendo apreciado por todos, favor observar e seguir nossas regras. Não toleraremos jogadores que incomodam e, se você violar nossas regras, provavelmente sua conta receberá uma suspensão ou restrição. Membros suspensos ou restringidos por violarem as nossas regras não têm direito a nenhum tipo de reembolso.
