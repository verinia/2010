Downloads
Os arquivos extras a seguir estão disponíveis para download.

Nota: Não é necessário fazer o download de nenhum destes arquivos para jogar RuneScape. Você pode jogar diretamente no seu navegador selecionando "Jogar RuneScape" na tela inicial!

Os arquivos a seguir são bônus extras oferecidos além do jogo aos jogadores que os desejar.

Estes são os únicos downloads oficiais do RuneScape. Downloads não oficiais podem conter vírus, keyloggers, etc., então se você deseja rodar o jogo fora de seu navegador, é recomendável manter sua conta segura, usando somente o cliente daqui.
Windows Client
Inicie o RuneScape a partir da sua área de trabalho. É muito útil especialmente para computadores com pouca memória ou baixa resolução. (É necessário ter Explorer 5.5 ou superior)
