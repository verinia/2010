O jogo não inicia




Se o RuneScape não inicia, trava ou está lento, tente uma das seguintes possíveis soluções:
  • Tente reiniciar seu computador.
    Se o RuneScape normalmente funciona para você mas não está funcionando hoje, a primeira coisa a fazer é simplesmente reiniciar seu computador. Isso muitas vezes resolve o problema!

  • Atualize o Java.
    Clique aqui para fazer o download e instalar a versão Java mais recente.

  • Faça o download das últimas correções do Windows.
    Clique aqui para visitar o Windows Update e instalar as últimas correções para o seu sistema operacional.

  • Tente mudar as configurações "Opções Java".
    Vá para a página Opções Java em nosso site (na caixa de seleção "Jogar agora") e tente alterar as configurações Java para "Sun Java Forte", pressione "Salvar configurações" e depois carregue o jogo normalmente.

  • Tente executar na versão mais recente do FireFox.
    Faça o download da versão mais recente do Firefox aqui e tente usá-lo para jogar o RuneScape.

  • Tente iniciar em "Modo Seguro".
    Enquanto o jogo estiver carregando (quando a barra de progresso estiver na tela), pressione a tecla "s" do seu teclado. Isso fará com que o jogo inicie no Modo Seguro.

  • Feche todos os programas desnecessários antes de executar o RuneScape.
    Feche todas as janelas e programas desnecessários antes de iniciar o RuneScape.
