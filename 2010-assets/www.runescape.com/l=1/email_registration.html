E-Mail-Hilfe

Sie können für Ihr Jagex-Konto bei der Kontoerstellung eine E-Mail-Adresse angeben. Auf dieser Seite finden Sie Informationen über die Registrierung und Änderung Ihrer E-Mail-Adresse und -Einstellungen.

E-Mail-Registrierung

Wenn Sie für Ihr Jagex-Konto eine E-Mail-Adresse angeben, können wir Sie gelegentlich bezüglich zukünftiger Updates kontaktieren und Ihnen die bestmögliche Kundenbetreuung zukommen lassen. Um sich für Updates per E-Mail einzuschreiben, klicken Sie bitte hier.


Wenn Sie Ihre E-Mail-Adresse später ändern möchten, können Sie das auf dieser Seite tun. Sobald Sie eine neue E-Mail-Adresse eingeben haben, wird ein Bestätigungscode an Ihre alte E-Mail-Adresse geschickt, damit wir sicherstellen können, dass Sie die Änderung wirklich selbst vorgenommen haben. Sie müssen außerdem einen Prüfcode eingeben, der an Ihre neue E-Mail-Adresse geschickt wird.

E-Mail-Optionen

Sie können die E-Mail-Optionen ändern und z.B. einen bestimmten Service abbestellen oder aktualisieren. Alle Änderungen müssen durch die Eingabe eines Bestätigungscodes verifiziert werden.



Code eingeben

Wenn Sie Ihre E-Mail-Adresse eingeben oder Ihre E-Mail-Optionen umstellen wollen, wird Ihnen ein Code zugeschickt, den Sie auf unserer Website eingeben müssen. Zu der Webseite, auf der Sie den Code eingeben müssen, gelangen Sie über den Link in unserer E-Mail oder über diesen Link. Sie müssen jeden Code nur einmal eingeben.
