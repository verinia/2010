Meldung von Spielfehlern


Klicken Sie hier, um einen Spielfehler zu melden

  1. Wie man einen Fehler meldet.
  2. Was versteht man unter einem Spielfehler?
  3. Wann sollte ich einen Spielfehler melden?
  4. Was versteht man unter Missbrauch von Spielfehlern?

1. Wie man einen Fehler meldet.

Einen Spielfehler können Sie melden, indem Sie hier klicken.

Wenn Sie das Fehlerhinweis-Formular ausfüllen, geben Sie uns bitte so viele Informationen und Details wie möglich. Das wird uns dabei helfen, den Fehler zu finden und ihn so schnell wie möglich zu beheben. Das Formular wird nach dem Versenden an einen Jagex-Mitarbeiter weitergeleitet, der Ihre Meldung bearbeiten wird. Bitte haben Sie Verständnis dafür, dass Jagex Spieler für Fehlerhinweise nicht belohnen kann und wir Ihnen auch nicht immer eine Antwort schicken werden.

Bitte senden Sie nur Spielfehler ein, die Sie selbst beobachtet haben, und keine Fehler, auf die Sie von Freunden oder im Forum aufmerksam gemacht wurden.

Ein Fehlerhinweis sollte möglichst detaillierte Informationen beinhalten, damit wir den Fehler reproduzieren können. Ein kleines Detail wie ein Ausrüstungsgegenstand, den Sie getragen oder im Inventar hatten, kann dazu führen, dass der Fehler auftritt oder eben nicht.

Ein Beispiel für einen Fehlerhinweis, der uns nicht weiterhilft, wäre: 'Ein Spieler steckt bei einem Baum in der Nähe von Varrock fest.'

Mit diesem Satz können wir weder den genauen Standort des Baums in Varrock ausfindig machen, noch können wir rekonstruieren, warum der Spieler dort feststeckt.

Idealerweise würde man diesen Fehlerhinweis so formulieren: 'Ein Spieler steckt ein Feld südlich von einem normalen Baum fest. Den Baum findet man, wenn man aus der Bank im Osten von Varrock rausgeht, die Treppen runter, dann neun Schritte Richtung Osten und zwei Schritte nach Norden. Bevor der betreffende Spieler feststeckte, hatte er gerade das Abenteuer 'Der Aushilfskoch' beendet und war dabei, einen Goblin mit einer Bronze-Streitaxt anzugreifen.'

Sie sollten das Fehlerhinweis-Formular nur für Fehler im Spiel selbst und auf unserer Webseite benutzen. Wenn sich Ihre Anfrage auf Abrechnungsprobleme, technische Dinge oder Werbung bezieht, benutzen Sie bitte die entsprechenden Links für: Mitgliedszahlung, Technisches, Werbung.

Wenn Sie dort keine Antwort auf Ihr Problem finden, dann kontaktieren Sie bitte die entsprechende Abteilung, um Hilfe anzufordern. Wir werden Fehlerhinweise, die keine richtigen Fehlermeldungen sind, einfach ignorieren. Weitere Informationen darüber, was ein Fehler ist, finden Sie hier.

2. Was versteht man unter einem Spielfehler?

Ein Spielfehler ist ein Stück falsch programmierter Code, das sich auf das Spiel auswirkt. Alles, was RuneScape daran hindert, so zu funktionieren wie es sollte, wird als Spielfehler angesehen. Ein Beispiel wäre, wenn Sie 10.000 EP statt der normalen 10 EP bekommen, wenn Sie eine gewöhnliche Handlung ausführen.

Wenn Sie meinen, dass Sie auf einen Spielfehler gestoßen sind, können Sie sich hier darüber informieren, wie man ihn meldet. Nutzen Sie den Spielfehler nicht dazu aus, um sich einen unfairen Vorteil zu verschaffen. Das Ausnützen von Spielfehlern verstößt gegen die Regeln und kann zu einer Kontosperrung führen.

Hier können Sie sich über den Missbrauch von Spielfehlern informieren.

Zurück zum Anfang

3. Wann sollte ich einen Spielfehler melden?

Benutzen Sie die Fehlermeldung nicht als Weg, um einfach mit Jagex in Kontakt zu treten.

Wenn Sie lediglich bei einem Abenteuer nicht weiterkommen, ist das ebenfalls kein Grund für einen Fehlerhinweis. Probieren Sie erst alle möglichen Optionen aus, bevor Sie einen Fehlerhinweis abschicken. Sprechen Sie mit allen Nicht-Spieler-Charakteren (NSC), die in dem Abenteuer vorkommen, um nach Hinweisen zu suchen, die Sie vielleicht übersehen haben. In einigen Fällen werden die NSC benötigte Gegenstände nicht aushändigen, wenn Sie sie bereits in Ihrem Bankschließfach oder Inventar haben, also sehen Sie erst dort nach. Wenn das nichts hilft, wenden Sie sich an Ihre Mitspieler im Spiel oder stellen Sie Ihre Frage ins Forum.

Schicken Sie uns erst dann einen Fehlerhinweis, wenn Sie ganz sicher sind, dass Sie alle Optionen ausgetestet haben. Bitte erinnern Sie sich daran, dass wir grundsätzlich nie verlorene oder gestohlene Gegenstände ersetzen werden.

Jagex duldet keinen Missbrauch des Fehlerhinweis-Formulars und wird gegen Spieler vorgehen, die das System ausnützen.

Zurück zum Anfang

4. Was versteht man unter Missbrauch von Spielfehlern?

Wenn ein Spieler einen Spielfehler entdeckt hat und ihn dazu verwendet, sich einen persönlichen Vorteil zu verschaffen, dann ist das ein Missbrauch von Spielfehlern. Sollten Sie auf einen Fehler stoßen, der Ihnen einen unfairen Vorteil verschaffen könnte, schicken Sie uns bitte umgehend einen Fehlerhinweis. Folgen Sie einfach dem Link Fehlerhinweis im Hilfs-Menü auf der RuneScape-Startseite oder klicken Sie hier.

Bitte beachten Sie, dass der Missbrauch von Spielfehlern gegen die RuneScape-Regeln verstößt. Wir werden Spieler bestrafen, die sich durch den Missbrauch eines Spielfehlers einen unfairen Vorteil verschaffen wollen.
