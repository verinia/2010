Le jeu ne démarre pas




En cas de non démarrage, d'arrêt soudain ou de lenteur d'exécution du jeu, essayez l'une des solutions suivantes :
  • Redémarrez votre ordinateur.
    Si RuneScape s'exécute normalement d'habitude, la première chose à faire est de redémarrer votre ordinateur. Cela règle souvent le problème !

  • Mettez Java à jour
    Cliquez ici pour télécharger et installer la dernière version.

  • Téléchargez les dernières mises à jour Windows.
    Cliquez ici pour installer les dernières mises à jour Windows de votre système d'exploitation.

  • Changez les options Java.
    Allez à la page Options Java de notre site (dans le menu déroulant de Jouer maintenant). Choisissez Forcer Sun Java, enregistrez votre choix, puis chargez le jeu normalement.

  • Utilisez la dernière version de FireFox.
    Vous pouvez télécharger la dernière version de FireFox ici.

  • Lancez le jeu en version recommandée.
    Lorsque le jeu est en cours de chargement (la barre de chargement est à l'écran) appuyez sur la touche « s » pour démarrer le jeu en version recommandée.

  • Fermez tous les programmes inutiles avant de lancer RuneScape.
    Fermez tous les programmes et fenêtres inutiles avant de lancer RuneScape.
